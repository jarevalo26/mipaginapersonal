Blog Personal

Tecnologías: HTML5, CSS3 puro

Descripción: Un blog personal completamente responsive creado desde cero utilizando únicamente HTML semántico y CSS puro. Incluye sistema de navegación, diseño adaptativo y optimización para diferentes dispositivos.

Características destacadas:

  • Diseño responsive sin frameworks
  • HTML semántico y accesible
  • CSS Grid y Flexbox para el layout
  • Optimizado para SEO

Estado: Completado

Aplicación de Gestión de Tareas

Tecnologías: JavaScript ES6+, LocalStorage, HTML5, CSS3

Descripción: Una aplicación web para gestión de tareas personales que permite crear, editar, eliminar y marcar tareas como completadas. Los datos se almacenan localmente en el navegador.

Características destacadas:

  • Interfaz intuitiva y moderna
  • Persistencia de datos con LocalStorage
  • Filtrado y búsqueda de tareas
  • Validación de formularios
  • Notificaciones y confirmaciones de usuario

Estado: Completado

Sitio Web Portfolio Responsive

Tecnologías: HTML5, CSS3, JavaScript, CSS Grid, Flexbox

Descripción: Un sitio web portfolio profesional completamente responsive que se adapta perfectamente a cualquier dispositivo, desde móviles hasta pantallas de escritorio grandes.

Características destacadas:

  • Mobile-first design approach
  • Animaciones CSS suaves
  • Optimización de imágenes
  • Cross-browser compatibility
  • Accesibilidad web (WCAG guidelines)

Estado: En desarrollo
